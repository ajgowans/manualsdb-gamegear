# The MiSTer Manuals DB - Sega Game Gear

This is a database for the MiSTer project that downloads the English manuals in .pdf form for the Sega Game Gear to the docs folder, which can be loaded from the OSD within th core by selecting "Help".

To use it simply copy and paste the below to the bottom of your downloader.ini file (found at: /media/fat/downloader.ini )

```ini
[ajgowans/manualsdb-gamegear]
db_url = https://raw.githubusercontent.com/ajgowans/manualsdb-gamegear/db/db.json.zip
```
Or download this .ini file and put it in /media/fat
https://github.com/ajgowans/manualsdb-gamegear/blob/db/downloader_ajgowans_manualsdb-gamegear.ini
